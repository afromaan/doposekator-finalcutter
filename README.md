# FinalCutter 

DIY Remote Controlled Lawn Mower

## Overview
FinalCutter is a DIY RC lawn mower with skid steering and welded frame.

## Features
- Gasoline engine (motor1name)
- 4 wheel drive motors (motor2name)
- FlySky remote control
- Raspberry Pi control
- Battery charging alternator

## Hardware
- Frame: DIY welded steel
- Engine: motor1name
- Drive motors: motor2name
